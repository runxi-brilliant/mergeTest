# 🎓 CampusTrade

A Secure, Hyper-Local Peer-to-Peer Marketplace for University Students

## 📖 Overview
CampusTrade is a web-based platform designed exclusively for university students to buy, sell, and exchange second-hand goods safely within their campus community.

## 🚀 Features
- 🔐 Secure authentication via university email
- 📦 Item listing and management
- 🔍 Advanced search and filtering
- 💬 Direct messaging between buyers and sellers
- 📱 Responsive design for mobile and desktop

## 🛠️ Tech Stack
- **Backend**: Python, FastAPI
- **Frontend**: React.js (Vite)
- **Database**: MongoDB
- **Authentication**: JWT

## 📦 Installation

### Backend Setup
```bash
cd backend
pip install -r ../requirements.txt
uvicorn main:app --reload
```

### Frontend Setup (Coming Soon)
```bash
cd frontend
npm install
npm run dev
```

## 🧪 Testing
```bash
pytest
```
